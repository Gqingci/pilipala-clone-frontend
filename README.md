# 🎬 pilipala-frontend

> 基于 Vue3 构建的在线视频分享系统前端项目  
> 实现视频播放、互动系统、创作中心及后台管理界面

---

## 📖 项目简介

pilipala Frontend 是在线视频分享系统的前端部分，基于 Vue3 + ElementPlus 构建，采用前后端分离模式，与 SpringBoot 后端进行接口交互。

项目实现完整的视频平台功能，包括视频浏览、互动、发布、数据统计和后台管理。

---

## 🛠 技术栈

- Vue 3
- Vite
- ElementPlus
- Pinia（状态管理）
- Mitt（事件总线）
- Axios

---
